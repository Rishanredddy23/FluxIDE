**FluxIDE** is a modern AI-powered development environment for Arduino and embedded systems.
It lets you generate, edit, and upload code to your microcontroller — all in one place.

## 🚀 Features

* 🤖 **AI Code Generation**
  Generate complete Arduino sketches using simple natural language prompts.

* ⚡ **One-Click Upload**
  Instantly upload generated code directly to your Arduino board.

* 🔌 **Arduino Integration**
  Built specifically for Arduino with support for `.ino` workflows and board selection.

* 🧠 **Smart Project System**
  Create, manage, and organize your embedded projects easily.

* 📡 **Serial Monitor (Coming Soon)**
  View real-time output from your microcontroller.

* 🖥️ **Modern UI/UX**
  Clean, fast, and intuitive interface inspired by professional IDEs.

---

## 🧪 Example Workflow

1. Type:

   ```
   Blink LED on pin 13 every 500ms
   ```
2. Click **Generate Code**
3. Click **Upload**
4. Done ✅ — Code runs on your Arduino

---

## 📦 Installation

1. Go to the **Releases** section
2. Download:

   ```
   FluxIDE Setup.exe
   ```
3. Install and launch the app
